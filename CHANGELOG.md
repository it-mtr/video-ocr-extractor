## [1.0.0] - 2025-11-22

### 新增
- ✨ 基于 PaddleOCR 的视频文字提取功能
- 🎯 精准的时间戳记录
- 🔍 Web 搜索界面（模糊查询）
- 📊 实时进度监控页面
- 📄 分页浏览所有记录
- ⚡ GPU/CPU 自动切换机制
- ⚙️ 可配置的识别参数（帧率、置信度）
- 🕒 时间偏移量修正功能
- 💾 SQLite 数据库存储
- 🌐 Flask Web 服务

### 技术栈
- Python 3.8+
- PaddleOCR 2.7+
- Flask 2.3+
- OpenCV 4.8+
- SQLite 3

## 版本说明

- **[未发布]**: 正在开发中的功能
- **[主版本号.次版本号.修订号]**:
  - 主版本号：不兼容的 API 修改
  - 次版本号：向下兼容的功能性新增
  - 修订号：向下兼容的问题修正

## 类型标识

- `新增`: 新功能
- `变更`: 已有功能的变更
- `废弃`: 即将移除的功能
- `移除`: 已移除的功能
- `修复`: 错误修复
- `安全`: 安全相关的修复
